# 🏢 Real Estate CRM Intelligence Dashboard

A data-driven **Machine Learning + Analytics Dashboard** built using Streamlit to analyze customer behavior, segment buyers, and understand investment patterns in real estate transactions.

---

## 🚀 Live Demo
👉 https://analytics-real-estate.streamlit.app/

---

## 📊 Project Overview

This project focuses on **real estate customer intelligence** by combining:
- Buyer segmentation
- Investment profiling
- Sales performance analytics
- Geographic insights
- Customer engagement behavior

The goal is to help real estate businesses:
- Identify high-value clients
- Understand buying behavior
- Improve targeting strategy
- Increase revenue efficiency

---

## 🧠 Key Features

### 📌 Customer Analytics
- Total clients overview
- Average property price analysis
- Engagement & satisfaction scoring

### 👥 Segmentation Analysis
- High-value vs mid vs budget buyers
- Segment-wise revenue contribution
- Behavioral comparison across groups

### 💰 Sales Intelligence
- Price distribution analysis
- Property size vs price relationship
- Revenue breakdown by segment

### 📈 Investment Insights
- Investment score profiling
- Buyer risk classification
- Cluster-based analysis

### 🌍 Geographic Insights
- Country-wise revenue distribution
- Regional performance breakdown
- Market comparison view
